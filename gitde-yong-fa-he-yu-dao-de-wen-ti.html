Git的用法和遇到的问题

2015-05-02

Git的用法其实很简单，首先你得注册一个Github的账户，一般要在你的Github上创建一个repo。 那么git就有用武之地了，下面我们来介绍下具体的步骤以及注意的事项。

经验之谈

就我经常用到的git步骤如下：

git add .
git commit -m "你更改的描述"
git push

其实配置好了后就是这么简单，有时候在不同机器上会用到git pull、git add --all...,但是百度一下你就会了，而且git的提示功能是很Bug的一个存在，任何错误都会提醒你用什么命令，所以不用担心出现错误了。

有一点需要注意的是，有时候可能需要在git前面加sudo，这个你懂的。

具体步骤

初始配置

  • 配置git: 刚申请git的时候，必须按如下设置配置username和useremail

git config --global user.name "you name"
git config --global user.email you@yourdomain.example.com
git config --global push.default simple(可以有其他选项)

  • 安装ssh公钥: 用你在注册github时用的Email生成一个ssh公钥对

ssh-keygen -t rsa -C "your_email@example.com"
