Markdown语法总结

Date 2015-04-16 Tags code / markdown

完成了github博客的搭建（稍后会介绍如何搭建），先学习一下Markdown。 Markdown是一个将文本转化为HTML的工具，就是为了方便我们编写网页显示文章。 (本文档仅在vim插件markdown中使用，其他的平台可能有一些偏差)
