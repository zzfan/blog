利用Pelican和Markdown搭建自己的github静态Blog

Date 2015-04-29 Tags blog / markdown

概述

从自己搭建Blog有一段时间了，其实为了方便我的Blog大部分都是从Github上荡下来的，有些地方没有著名版权，敬请谅解。

动态的Blog我也弄过一段时间，是用node.js开发的，当时也是由于没有找到合适的免费VPS（用过一段时间的 DigitalOcean，也就没有继续下去了（对于学生来说还是有点贵的）。如果有需要，可以关注我的 Github。

经过一段时间的摸索，完成了我的Github Page: http://blog.zzfanhero.info。

其实这个 轩枫阁才是我一直想要搭建的Blog（很喜欢那种自己开发的可以不断自己的完善，而且有无限大的开发空间），但是我的方向不再网页开发，只能忍痛不去想那绚丽的让人眼花缭乱的Blog了\~\~

具体搭建

这里我就不很详细的列举出其中的细节了，直接上命令，是在Linux上搭建的。

准备工作

  1. github: 首先你得会用github
  2. markdown:　用这个写blog比较舒服
  3. linux:　我的所有工作都是再linux下完成的
  4. js:　所有的编程语言都是想通的，对于模板的一些更改来适应自己的风格

安装软件

  1. pelican: 自己去百度
  2. git: 自己去配置

Let't go

mkdir blog
cd blog
pelican-quickstart: 指定域名（xxx.github.io）

这时你会看到blog目录下生成了一些配置文件，主要要更改的就是pelicanconf.py

开始写blog

Date: xxx-xx-xx
Title: ...
Tags: life,code
Gategory: Code
....Content....

cd blog
sudo make html

然后你就可以再你的github上新建一个blog工程了，git到本地，然后把output中的内容拷到blog里面就可以了!

cd blog/blog
git add --all
git commit -m ""
git push

过一会访问xxx.github.com就行了．（xxx就是你的github用户名） 如果你有域名的话，比如你在Godady上买了域名，可以在Godaddy上配置主机为blog.域名指定xxx.github.io,然后在你的项目根目录下创建"CNAME"文件，写入你的域名．

其他

  • 主题的更改

git clone git://github.com/getpelican/pelican-themes.git
sudo pelican-themes -i $(blogpath)/pelican-themes/bootstrap2
pelican-themes --list --verbose
make html

  • 插件的添加

git clone git://github.com/getpelican/pelican-plugins.git

  • 添加评论的支持
    见配置文件

  • 拷贝静态文件
    见配置文件
