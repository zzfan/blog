二叉树的遍历及应用

Date 2016-03-24 Tags 数据结构 / 面试

摘要：先中后序遍历，深度广度，层次，路径，每层节点数。

* 二叉树的遍历

由于二叉树是一种递归结构，所以我们很容易用递归方式实现其遍历操作。

1. 先序遍历

（1）访问根节点； （2）先序遍历左子树； （3）先序遍历右子树； 

void PreOrder(Bitree T){
    if(!T) return;
    else{
        visite(T->data);
        PreOrder(T->lchild);
        PreOrder(T->rchild);
    }
}

2. 中序遍历

（1）先序遍历左子树； （1）访问根节点； （3）先序遍历右子树； 

 void InOrder(Bitree T){
     if(!T) return;
     else{
        InOrder(T->lchild);
         visite(T->data);
         InOrder(T->rchild);
     }
 }

3. 后序遍历

（1）先序遍历左子树； （2）先序遍历右子树； （3）访问根节点； 

void PostOrder(Bitree T){
    if(!T) return;
    else{
        PostOrder(T->lchild);
        PostOrder(T->rchild);
        visite(T->data);
    }
}

4. 层序遍历二叉树

（1）初始化一个空队列；
（2）非空根指针入队。
--队头元素出对，访问之； --若被访节点有左孩子，则左孩子入队； --若被访节点有右孩子，则右孩子入队；

void LayerTraversal(Bitree T){
    InitQueue(Q);
    if(T) EnQueue(Q, T));
    while(!QueueEmpty(Q)){
        DeQueue(Q, p);
        visite(p->data);
        if(p->lchild) EnQueue(Q, p->lchild);
        if(p->rhicld) EnQueue(Q, p->rchild);
    }
}

显然，对二叉树的各种算法中，基本操作是访问节点。不论按哪种次序遍历含有n个节点的二叉树，其时间复杂度均为O(n).

* 其他运算

记住：递归是二叉树的固有特性，采用递归的方式来实现二叉树的有很多操作或运算就非常自然而且简单。

1. 求二叉树的节点个数

其实利用任何一种遍历方式都可以，下面我们先介绍两种方案。
（1）利用二叉树的递归特性。若二叉树为空，则节点数为0；否则
　　二叉树节点个数　＝　左子树节点个数　＋　右子树节点个数　＋　１个根节点

int CountNodes1(Bitree T){
    if(!T) return 0;
    else{
        n1 = CountNodes1(T->lchild);
        n2 = CountNodes2(T->rchild);
        return(1 + n1 + n2);
    }
}

（２）利用先序遍历计数 

void CountNode2(Bitree T){
    if(!T) return 0;
    else{
        n++;
        CountNodes2(T->lchild, n);
        CountNodes2(T->rchild, n);
    }
}

2. 输出二叉树每个节点的层次数

void Level(Bitree T){

3. 求二叉树深度和宽度

（1）二叉树深度

int Depth(Bitree T){
    if (!T) return 0;
    else{
        hl = Depth(T->lchild);
        hr = Depth(T->rchild);
        return(hl>hr ? hl+1:hr+1);
    }
}

（1）二叉树宽度 

//求二叉树的宽度  
int GetWidth(tagBiNode *pRoot)  
{  
    if (pRoot == NULL)  
    {  
        return 0;  
    }  

    int nLastLevelWidth = 0;//记录上一层的宽度  
    int nTempLastLevelWidth = 0;  
    int nCurLevelWidth = 0;//记录当前层的宽度  
    int nWidth = 1;//二叉树的宽度  
    queue<BiNode *> myQueue;  
    myQueue.push(pRoot);//将根节点入队列  
    nLastLevelWidth = 1;      
    tagBiNode *pCur = NULL;  

    while (!myQueue.empty())//队列不空  
    {  
        nTempLastLevelWidth = nLastLevelWidth;  
        while (nTempLastLevelWidth != 0)  
        {  
            pCur = myQueue.front();//取出队列头元素  
            myQueue.pop();//将队列头元素出对  

            if (pCur->left != NULL)  
            {  
                myQueue.push(pCur->left);  
            }  

            if (pCur->right != NULL)  
            {  
                myQueue.push(pCur->right);  
            }  

            nTempLastLevelWidth--;  
        }  

        nCurLevelWidth = myQueue.size();  
        nWidth = nCurLevelWidth > nWidth ? nCurLevelWidth : nWidth;  
        nLastLevelWidth = nCurLevelWidth;  
    }  

    return nWidth;  
}

总结：二叉树的操作大多是递归，层次遍历的用了一个队列，所以说还是比较简单的，求宽度的时候就是层次遍历的一个拓展
